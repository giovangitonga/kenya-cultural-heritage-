## рџ“– Overview

Kenya Cultural Heritage Hub is a static web application that introduces users to the four major ethnic language families found in Kenya:

* **Bantu** (Kikuyu)
* **Nilotes** (Luo)
* **Cushites** (Borana)
* **Semites** (Arab/Swahili)

The platform provides historical background, migration patterns, cultural traditions, governance systems, and community lifestyles while encouraging learning through interactive quizzes.
